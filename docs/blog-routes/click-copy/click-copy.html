Expanded Clipboard Webpage

A mutable, click-to-copy webpage skeleton for all your expanded clipboard needs!

I made an extremely lightweight tool for those who want an explanded and fully visible clipboard. I found myself needing to copy-and-paste a lot of information during the internship application seasons, and rather than constantly highlighting and copy-pasting between windows, I found it much faster to just click! This tool also condenses all the copy-able text into single text bars instead of whole text blocks, which minimizes the distance your mouse needs to travel. Text bars are editable and work on the go!

This webpage is local host for all to use.

A how-to-use and source code can be found here!

Example of a filled-out click-to-copy

Another very lightweight tool that I coded with this repo is an "enter-to-copy". Useful for quickly replacing a few words within a largely templated response (for example, cover letters)! All I needed to do was to change 'brand' and then run the cell block.
