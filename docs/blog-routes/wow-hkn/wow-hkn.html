Multi-Boxing Control Interface

An old multi-boxing script that I used to play World of Warcraft: Wrath of the Lich King!

For those unfamiliar with World of Warcraft (WoW), it is a massive online multiplayer game where you controlled a character and went on adventures. There was one niche gameplay element within in game called "multi-boxing", which is using a control interface to control multiple characters at once. This allowed the solo player to emulate a perfectly coordinated multi-player team, providing that solo player with a number of ingame advantages: from time saves to better resource collection.

These scripts were created using Notepad originally in 2017 or so.

The associated repo contains two scripts: an AutoHotKey (AKN) script and a HotKeyNet (HKN) script. I wrote and used both, back when I played the game. For anyone that is still playing and want to see how to program their own scripts, the repo can be found here!
